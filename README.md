# Split by Column - Desktop Data Splitter (Enhanced MVP)

A **Tkinter-based desktop application** for splitting Excel or CSV datasets by column values. Designed for data professionals, this tool allows you to:

- Load Excel (`.xlsx`, `.xls`) or CSV files
- Auto-detect all columns
- Select one, multiple, or all columns for splitting
- Live, always-visible preview of selected columns, unique groups, and sample output filenames
- Export split data as CSV or Excel
- Automatically bundle split files into a ZIP archive
- Track progress with a visual progress bar
- View detailed activity logs
- Robust error handling for safe operation

---

## **Features**

✅ Tkinter GUI with modern layout  
✅ Dynamic, auto-updating preview section  
✅ Smart filenames based on column values  
✅ Multi-column selection with "Select All" option  
✅ Export as CSV or Excel  
✅ Automatic ZIP archive creation  
✅ Progress tracking and logs  
✅ Error handling and safe filename sanitization  
✅ Reset and exit buttons
